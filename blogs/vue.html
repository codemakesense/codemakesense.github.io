# Vue

# 生命周期钩子

# beforeCreate：

这个时候 Vue 实例刚刚初始化，实例中的 data 和 methods 还不能被访问

# created：

这个时候实例中的 data 和 methods 已经初始化，能够访问，但是页面的 dom 元素还在内存里，未被渲染到页面中，因此无法获取 dom 中的元素

# beforemount：

这个时候实例已经将 dom 元素渲染到页面中，但是还未将页面数据更新，页面上的数据还是初始的，实例未挂载到页面的容器中

# mounted：

这个时候实例已经挂载到页面的容器中， 页面中的数据已经是最新的

# beforeUpdate：

在实例挂载到页面容器后，如果 data 需要进行更新操作，那么会先进入

# beforeUpdate

生命周期，在这个阶段，Vue 的 data（model 层）已经更新，但是 view（视图层）仍是旧的 > updated：

在这个阶段，Vue 实例已经完成虚拟 dom 树的数据更新并将最新的 dom 树渲染到视图层中

# beforeDestory：

在实例销毁阶段，实例中的 data 和 methods 等属性，监听器，管道器，指令等仍能访问

# destoryed：

实例完成销毁过程，实例中所有的 data，methods 等所有属性都不可用
